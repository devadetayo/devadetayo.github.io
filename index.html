Quanta CSS

My Very Own CSS Framework

Quanta CSS is my favorite project — a lightweight, utility-first CSS framework I built to speed up UI development and keep styles consistent across projects.

  • Utility-first: Fast to use with concise classnames for common layout, spacing, and typography needs.
  • Responsive grid: Lightweight, mobile-first grid system that adapts across breakpoints.
  • Theming & accessibility: Built with dark-mode support and accessible defaults for better UX.

Available as a standalone CSS file — ideal when you want the speed of utility classes without the overhead of a large dependency.
